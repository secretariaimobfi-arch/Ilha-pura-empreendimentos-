# Ilha Pura Empreendimentos

Repositorio para organizar os materiais publicos encontrados no Linktree de origem:

- Linktree: https://linktr.ee/ipanamahfuz
- Perfil Instagram: https://instagram.com/anamahuz
- Perfil LinkedIn: https://www.linkedin.com/in/ana-mahfuz-2b987b30

## Escopo

Este repositorio cataloga os materiais visuais e documentos disponiveis publicamente nos folders do Google Drive ligados ao Linktree. Os arquivos grandes continuam referenciados no Drive por links de visualizacao e download para evitar duplicacao pesada no GitHub.

## Arquivos do repositorio

- `sources.json`: manifesto com os links dos folders, contagens e links de disponibilidade.
- `docs/links.md`: links organizados para consulta rapida.
- `scripts/download-from-drive.ps1`: script base para baixar arquivos quando os links publicos estiverem acessiveis.
- `assets/*/.gitkeep`: estrutura de pastas por empreendimento.

## Observacao operacional

Os arquivos listados vem de fontes externas. Antes de republicar imagens, videos ou PDFs diretamente no GitHub, confirme permissao de uso e distribuicao. Para uso interno, manter os links do Drive costuma ser mais pratico e evita limites de tamanho do GitHub.
